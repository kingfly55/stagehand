2. **Ensure target directory exists:**

3. **Move each planning document** using `git mv` to preserve history:
   ```bash
   for f in PLAYWRIGHT_NATIVE_PLAN.md PHASE4_FINAL_BRIEF.md PHASE4_ADDENDUM.md \
            V2_vs_V3_NATIVE_COMPARISON.md SNAPSHOT_UPGRADE_RESEARCH.md \

            harness-engineering.md AGENTIC_ENGINEERING_REQUIREMENTS.md; do
     if [ -f "$f" ]; then
       git mv "$f" docs/plans/completed/
     else
       echo "SKIP: $f not found"
     fi
   done
   ```
   **Note:** Using a loop with existence check prevents `git mv` from failing on missing files.
   **Note:** The pre-commit hook from Milestone 3 will run on this commit. Since these are `.md` files (not protected paths), it should pass. If typecheck fails for unrelated reasons, use `git commit --no-verify` and note the reason.

## Verification Steps
1. Check no planning docs at root:
   ```bash
   # List all root .md files excluding known keepers
   remaining=$(ls -1 *.md 2>/dev/null | grep -Ev '^(README|CHANGELOG|TESTING_GUIDE|claude)\.md$' || true)
   if [ -z "$remaining" ]; then
     echo "OK: no planning docs at root"
   else
     echo "FAIL: unexpected .md files at root: $remaining"
   fi
   ```
2. Check completed plans directory:
   ```bash
   # Use find instead of ls to avoid parsing issues
   count=$(find docs/plans/completed -maxdepth 1 -name '*.md' 2>/dev/null | wc -l)
   echo "Found $count files"
   # Should be >= 10 (9 moved files + INDEX.md)
   [ "$count" -ge 10 ] && echo "OK" || echo "FAIL: expected >= 10, got $count"
   ```
3. Confirm index files exist:
   ```bash
   test -f docs/plans/completed/INDEX.md && echo "OK" || echo "FAIL"
   test -f docs/plans/active/INDEX.md && echo "OK" || echo "FAIL"
   ```
4. Confirm template exists and has required sections:
   ```bash
   grep -c "Problem Statement\|Scope\|Approach\|Verification\|Acceptance Criteria" docs/plans/TEMPLATE.md
   # Should be >= 5
   ```
5. Confirm `claude.md` references plans:
   ```bash
   grep -q "docs/plans" claude.md && echo "OK" || echo "FAIL"
   ```
6. Confirm no stale root-level references remain in `claude.md`:
   ```bash
   # Check that all references to moved files include their new docs/plans/ path
   # Grep for any mention of the moved file names, then verify each has the docs/plans/ prefix
   stale=0
   for name in PLAYWRIGHT_NATIVE_PLAN PHASE4_FINAL_BRIEF PHASE4_ADDENDUM \
               V2_vs_V3_NATIVE_COMPARISON SNAPSHOT_UPGRADE_RESEARCH \
               AGENT_ORCHESTRATION DOCS_UPDATE_PLAN harness-engineering \
               AGENTIC_ENGINEERING_REQUIREMENTS; do
     # Find lines mentioning this file but NOT with docs/plans/ prefix
     if grep -n "$name" claude.md | grep -v "docs/plans/" | grep -q .; then
       echo "FAIL: stale reference to $name without docs/plans/ prefix"
       stale=1
     fi
   done
   [ "$stale" -eq 0 ] && echo "OK: no stale references" || echo "FAIL: stale references found"
   ```
7. Run `pnpm typecheck` and confirm zero errors
